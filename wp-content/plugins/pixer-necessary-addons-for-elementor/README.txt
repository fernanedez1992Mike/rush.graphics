== Description ==

This is Pixer Necessary addons plugin for elementor pagebuilder. With Pixer you can easily customize your website with premade Pixer elements.
Here in Pixer you will find 
BANNER ELEMENTS,
FEATURES ELEMENTS,
CONTENT WITH IMAGE ELEMENTS,
CONTENT WITH VIDEO ELEMENTS,
BRAND ELEMENTS,
PRICING ELEMENTS,
TEAM ELEMENTS,
NEWSLETTER,
VIDEO SECTION,
INFO BOXS & MORE COMING.....

[Check the Demos](https://themepixer.com/pixer)



== Installation ==
1. Upload `pixer=necessary-addons-for-elementor.zip` to the `/wp-content/plugins/` directory and exctract
2. Activate the plugin through the 'Plugins' menu in WordPress
